# 🏦 Bank Customer Churn Dashboard - Power BI

## 📌 Overview

This project presents an **interactive Power BI dashboard** designed to analyze bank customer churn behavior.

The dashboard provides clear insights into customer activity, financial behavior, and churn patterns to help businesses improve customer retention.

---

## 🎯 Objectives

* Analyze customer churn behavior
* Identify high-risk customer segments
* Understand factors affecting churn
* Support data-driven decision making

---

## 📊 Dataset

The dataset contains detailed information about bank customers, including:

* Customer ID
* Gender
* Geography
* Age
* Balance
* Credit Score
* Number of Products
* Has Credit Card
* Active Member
* Exited (Churn Status)

📁 File: `data/Churn_Modelling.csv`

---

## 📈 Key KPIs

* 👥 Total Customers: **1641**
* 🟢 Active Customers: **837**
* 💰 Average Balance: **121.18K**
* 📉 Churn Rate: **8%**
* 📦 Average Products per Customer: **1.56**

## 🧠 Key Insights

* Customers from certain regions show higher churn rates
* Low engagement customers are more likely to churn
* Higher balance customers tend to be more stable
* Active members have significantly lower churn
